# Internship Tracker API

A backend API built using Django REST Framework to track internship applications.

## Features
- Add internship applications
- View all applications
- Update status (Applied, Interview, Rejected, Accepted)
- Delete entries

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite

## API Endpoints
- GET /api/internships/
- POST /api/internships/
- PUT /api/internships/{id}/
- DELETE /api/internships/{id}/
